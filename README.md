# Proyecto: ¿Merece la pena el rediseño?
    Análisis del A/B Test de Vanguard

# Problema de negocio
Vanguard, empresa de inversiones, rediseñó su interfaz web con el objetivo de mejorar la experiencia de sus clientes y aumentar la tasa de finalización del proceso online.
Este proyecto analiza si el nuevo diseño consigue ese objetivo de forma estadísticamente significativa y si justifica económicamente la inversión realizada.

# Objetivo del proyecto
El objetivo de este proyecto es evaluar el impacto del nuevo diseño de la interfaz web de Vanguard a través de un experimento A/B, estudiando:

- Completion Rate → proporción de usuarios que completan el proceso
- Coste-efectividad → si la mejora supera el umbral mínimo del 5% establecido por Vanguard
- Tiempo entre pasos → si el nuevo diseño cambia el comportamiento del usuario
- Perfil de clientes → si ambos grupos son comparables en términos demográficos y financieros

La idea principal es determinar si el nuevo diseño mejora la experiencia del usuario de forma real y suficiente para justificar su implementación definitiva.
